# 🔥 Firebase Setup Guide — SMS REX

## Step 1: Firebase Account Banao (FREE)
1. Browser mein jao: https://console.firebase.google.com
2. Google account se login karo
3. "Create a project" click karo
4. Project name: **sms-rex** → Continue
5. Google Analytics: OFF karo → Create Project

## Step 2: Android App Add Karo
1. Project open karo → Android icon click karo
2. Package name: **com.smsrex.app**
3. App nickname: SMS REX
4. Register App click karo
5. **google-services.json** download karo
6. Yeh file rakho: `android/app/google-services.json`

## Step 3: Authentication Enable Karo
1. Left menu → Build → Authentication
2. "Get started" click karo
3. "Email/Password" enable karo → Save

## Step 4: Firestore Database Enable Karo
1. Left menu → Build → Firestore Database
2. "Create database" click karo
3. "Start in test mode" select karo → Next
4. Location: asia-south1 (India) → Enable

## Step 5: Firebase Cloud Messaging (FCM)
1. Left menu → Project Settings (gear icon)
2. Cloud Messaging tab
3. Server Key copy karke safe rakh lo
